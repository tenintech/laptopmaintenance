<h1>Laptop Hardware Maintenance Lab</h1>
<h2>Lab Objectives</h2>
Inspect internal laptop components and cooling system.
Replace thermal paste to improve heat transfer and system cooling.
Diagnose and resolve a non-functioning keyboard key through driver reinstallation.
Repair physical issues affecting charging stability.
Practice safe laptop disassembly and reassembly procedures.
In this tutorial, we observe various network traffic to and from Azure Virtual Machines with Wireshark as well as experiment with Network Security Groups. <br />


<h2>Video Demonstration</h2>

- ### [YouTube: Azure Virtual Machines, Wireshark, and Network Security Groups](https://www.youtube.com)

<h2>Environment</h2>

- Personal laptop (Windows-based system)
- Local administrator access
- Manufacturer drivers and system utilities

<h2>Tools & Materials Used </h2>

- Precision screwdriver set
- Thermal paste
- Cleaning materials (compressed air / lint-free cloth/ IPA)
- Replacement case screws
- Device Manager and system drivers
